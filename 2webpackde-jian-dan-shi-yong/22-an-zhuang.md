这里用node-npm来安装webpack
- 新建工程目录-WebPackDemo
- 在工程目录中初始化配置npm init，生成package.json文件
```
{
  "name": "webpackdemo",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "",
  "license": "ISC"
}
```
- 安装webpack
```
npm install webpack --save-dev
```
