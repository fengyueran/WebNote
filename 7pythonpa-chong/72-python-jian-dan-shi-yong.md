**函数**
声明函数用def
```
def 函数名(参数1，参数2，...，参数n)：

    函数体
    
//冒号的意思就是下面好开始真正的函数内容了,不能少
def myFunction(a,b):
    c = a + b
    print c

if __name__=="__main__":
    myFunction(1,2)
```
